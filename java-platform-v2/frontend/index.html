API Documentation

Base URL:

http://localhost:8080/api/v1

Authentication:

All endpoints require JWT token in Authorization header:

Authorization: Bearer <token>

Main Endpoints:

  • POST /auth/register - Register new user
  • POST /auth/login - Login user
  • POST /compile/run - Compile and run Java code
  • GET /compile/history - Get execution history
  • POST /chat/send - Send chat message
  • GET /chat/latest - Get latest messages
  • POST /ai/analyze-error - Analyze compilation error
  • GET /health - Health check endpoint

WebSocket Endpoints:

  • ws://localhost:8080/ws/chat - Real-time chat
